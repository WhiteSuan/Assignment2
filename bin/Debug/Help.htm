Describe the use case for inserting a record in the booking table:

Precondition: the users (customer/hotel staff) has downloaded and installed the hotel software into their PCs.

Actor action:

1. User double-clicks on the software to start it.

2. User fills in all booking information in the booking form, including […].

3. User clicks “Insert” button to save the booking information. The system then captures all the booking information and save it into the hotel database (both in the booking and invoice table). The system then generates the invoice and send to the customer. If the user is the customer, the system will send a notification to the hotel staff about this new reservation.

Alternatives course of event:

Line 3: system checks if the customer did fill all the fields in the booking form. If not, indicate error.

Inserting a record in the Booking table

1. Validation

Validation Room Type combobox:

If there is no item chosen in this combobox then

                All fields are not valid

                Displaying error picture

                Set message error in the picture as “Please select room number in Room Number combobox”

Else

                Hiding error picture

End if

 

Validation Customer ID combobox:

If there is no item chosen in this combobox then

                All fields are not valid

                Displaying error picture

                Set message error in the picture as “Please select a customer in Customer ID combobox”

Else

                Hiding error picture

End if

 

Validation Num of Days textbox:

Test if Value in Num of Days textbox is numeric or not with isNumericVal function

If not numeric than

All fields are not valid

                Displaying error picture

                Set message error in the picture as “Please enter number of days in numeric type”

Else

                Hiding error picture

End if

 

Validation Num of Guests textbox:

Test if Value in Num of Guests textbox is numeric or not with isNumericVal function

If not numeric than

All fields are not valid

                Displaying error picture

                Set message error in the picture as “Please enter number of guests in numeric type”

Else

                Hiding error picture

End if

 

2. Calculate Total price of this booking

Try

If Num of Days textbox has already been inserted a value and Room Number combobox has been chosen an item then

                Make a new connection to database

                Open database connection

                Select price in room table where room number = value in Room Number combobox

                Total price = price of that room number x number of days

End if

Catch Exception

                If there is any exception (error) when calculating total price then display the appropriate error message.

Finally

                Close database connection

End try

3. Assign values in the form to Hashtable data

If all fields are valid then

                Make a new connection to database

                Make new list of hashtable

                Make new data in hashtable

                Then assign all values in Booking form except total price and comment (including checking day, room number, customer ID, num of days, num of guests, booking dates) to datas of Hashtable

                If total price value has been calculated and displayed in Total Price text box then

                                Get the number in the total price value, not the “$”

                                And assign that number to data in hashtable

                Check if Comment textbox has value inside or not with isNullOfEmpty function

If this textbox has no value then

                Assign Null value to data in hashtable

Else

                Assign the value inside Comment textbox to data in hashtable

After all values in the Booking form has been assign to hashtable datas then run Insert function

End if

Else if all fields are not valid then

                Displaying message box “One of the fields was invalid”

End if

4. Insert function

Make a new connection to database

Try

                Open database connection

                Take the datas of Hastable and insert into Booking table in the approriate column of the table (for example, take “room number” data of Hastable and insert into room_number column of Booking table in database)

                Displaying message box “The record was inserted”

                Then update the availability of that room number to “No”

Catch Exception

                If there is any exception (error) when inserting datas into database then display the appropriate error message.

Finally

                Close database connection

End try
